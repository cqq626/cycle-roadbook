# 骑行路书生成工具

[体验地址](https://cqq626.github.io/cycle-roadbook/)

基于百度地图，打点生成骑行路线，将路线导出为 gpx 文件内容

## 功能

- [x] 右键打点生成骑行路线
- [x] 拖拽途经点重新规划路线
- [x] 途经点前后插入新的途经点 & 删除当前点
- [x] 右键选择生成 gpx 文件内容
- [x] 基于起点拉测距圆
- [x] POI/道路搜索
- [x] 当前定位
- [x] 街景查看
- [ ] 加载 strava 赛段数据
- [ ] 本地存储历史路线
